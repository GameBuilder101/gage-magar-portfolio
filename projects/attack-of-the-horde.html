    Description

    Attack of the Horde is a zombie-themed, arcade-style shooter inspired by Space Invaders and an 80's B movie tone. It was created for a class project in which we were required to develop an interactive web experience with sound and visuals using JavaScript. It served as a learning experience for JavaScript programming and the PixiJS library.

    Process & Challenges

    The first step of development was to propose an idea for the design. A major personal goal was to make this game concept simple enough to build quickly, but interesting enough to keep the player engaged. After this, the development process occurred in parallel with learning the PixiJS library by exploring the documentation.
