    Description

    Beneath the Permafrost is a first-person survival-horror game set in a frozen city. This project was originally developed solo for several years. In summer 2025, I partnered with an environment artist to register the project for the RIT MAGIC Maker program, serving as an academic co-op. The program was a major learning experience for managing team workflow and scope.

    Process & Challenges

    This has been under development for several years, going through many rewrites. During the Maker program, I had to massively reduce scope (a neglected problem from when it was a hobby project) to make both short-term and long-term timelines feasible. This project has also taught me a lot regarding game design, such as teaching players mechanics, guiding through level design, and UX.
