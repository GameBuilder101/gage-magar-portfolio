    Description

    Beneath the Permafrost is a first-person survival-horror game set in a frozen city. This project was originally developed solo for several years. In summer 2025, I partnered with an environment artist to register the project for the RIT MAGIC Maker program, serving as an academic co-op. The project has been a learning experience in terms of managing scope and bringing on new people to a team.

    Process & Challenges

    This has been under development for several years, going through many rewrites. During the Maker program, I had to massively reduce scope to make both short-term and long-term timelines feasible. This project has also taught me a lot regarding game design, such as onboarding players to gameplay mechanics, guiding through subtle cues level design, and creating immersive scenarios.
