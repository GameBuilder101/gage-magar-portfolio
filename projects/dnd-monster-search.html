    Skills Applied

    Description

    This is a small web page which allows the user to search through a database of monsters from the game Dungeons & Dragons. It was created as an academic project where the goal was to create an application which interfaces with an online API using a search system.

    Process & Challenges

    The project began by creating a sketch of the page layout. This enabled me to determine ahead of time which filters would be presented to the user and what information would be displayed in search results. After this, time had to be spent analyzing API documentation. During development, a challenging bug cropped up which turned out to be an issue with the API server.
