    Description

    Beneath the Permafrost is a first-person survival-horror game set in a frozen world. This is a long-running project developed solo (so far) using the Unity engine. It has played a major role in expanding my technical/creative skills and ability to structure a large game project.

    Process & Challenges

    This has been under development for several years. It went through a few story rewrites, as previous versions hinged too heavily on gameplay that was difficult to test and potentially frustrating. Some changes served as lessons that gameplay and technical aspects should be finished before adding visual refinement. More recently, emphasis has been placed on iterating upon mechanics and refining tutorials to feel natural.
